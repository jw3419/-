## 2. Project

### 🎈 Smart Airport가 무엇인가요?

Smart Airport는 공항 내의 시설을 추천해주는 서비스입니다.

인공지능을 통해 사용자가 선택한 시설과 유사한 시설들을 추천하여,<br>
인천공항내에 다양한 시설들을 폭 넓게 이용할 수 있도록 도움을 주는 웹 어플리케이션입니다.<br>
